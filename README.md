# ai-job-tracker

A simple web app to track job applications and prioritize them using a scoring system (deadline urgency + skill match).

## Tech Stack
- FastAPI
- SQLite (dev)
- SQLAlchemy
- scikit-learn (TF-IDF)

## Features (MVP)
- Create / update / delete applications
- Track status (Wishlist / Applied / Interview / Offer / Rejection)
- Priority score per application
